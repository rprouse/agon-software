# More BASIC Computer Games

Computer games from the classic book
[More BASIC Computer Games](https://archive.org/details/More_BASIC_Computer_Games)
by David H. Ahl.

Downloaded from [Roug Connect](https://www.roug.org/retrocomputing/languages/basic/morebasicgames)

## Converting

I am converting these files one at a time. If it is converted, there will be
a `*.BBC` version of the game.

If you convert any of the games, please create a pull request with the
updated `*.BAS` file and the tokenized `*.BBC` file.
